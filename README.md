# React Micro Frontend compiler(cli)
React, Typescript, Webpack 5, ModuleFederation


## Install
<blockquote>
  npm install @reactmf/compiler --save-dev
</blockquote>
Or
<blockquote>
  yarn add @reactmf/compiler --dev
</blockquote>

## How to use
```typescript
  import Micro from "@reactmf/compiler";
```
